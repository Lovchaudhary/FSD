🌟 EXAMCELL SETUP GUIDE (HINDI + SUPER SIMPLE) 🌟

Is guide ko bilkul basic level pe likha gaya hai.
Step by step follow karo, sab chal jayega.

⸻

🟣 STEP 1: CHECK KARO COMPUTER READY HAI YA NAHI

🔹 Node.js check karo:
	1.	Terminal / Command Prompt kholo
	2.	Type karo:
node -v
	3.	Enter dabao

👉 Result:
	•	Version number aaya (v18, v20, etc) → ✅ Node installed
	•	Error aaya → ❌ Install karo: https://nodejs.org

🔹 npm check karo:
npm -v
	•	Version aaya → ✅ OK
	•	Error → ❌ Node properly install nahi hua

⸻

🟣 STEP 2: MONGODB CHECK

🔹 MongoDB installed hai ya nahi:
mongod –version
	•	Version dikhe → ✅ Installed
	•	Error → ❌ Install karo: https://www.mongodb.com

🔹 MongoDB run karo:
mongod
	•	Bahut saare logs aaye → ✅ Mongo running

⚠️ IMPORTANT: Is terminal ko band mat karo

⸻

🟣 STEP 3: BACKEND START KARO (BRAIN)
	1.	Terminal kholo
	2.	Backend folder me jao:
cd path/to/backend
	3.	Install dependencies:
npm install
	4.	Server start karo:
npm start

👉 Expected output:
	•	ExamCell backend running on http://localhost:5000
	•	MongoDB connected

→ ✅ Backend ready

⸻

🟣 STEP 4: FRONTEND START KARO (WEBSITE)
	1.	Naya terminal kholo
	2.	Frontend folder me jao:
cd path/to/frontend
	3.	Install dependencies:
npm install
	4.	Start app:
npm start

👉 Expected:
	•	Browser automatically open hoga
	•	URL: http://localhost:3000

→ ✅ Website ready

⸻

🟣 FINAL CHECK LIST

✔ node -v → version
✔ npm -v → version
✔ mongod –version → version
✔ mongod → running logs
✔ backend npm start → server running
✔ frontend npm start → website open

⸻

⚠️ COMMON PROBLEMS

❌ command not found → install nahi hai
❌ port already in use → koi aur app chal raha hai
❌ npm install fail → internet ya dependency issue
❌ Mongo error → setup ya path issue
